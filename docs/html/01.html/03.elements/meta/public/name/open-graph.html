meta property="og:*" Open Graph 示例

本页面的 <head> 中包含一组 Open Graph 元数据，用于社交平台（如微信、QQ、Facebook、LinkedIn 等）在分享时生成标题、描述与封面图预览。

要点

  • og:title / og:description：主要文本信息。
  • og:image：清晰、合适尺寸的预览图，可多张。
  • og:url：页面规范 URL，避免重复。
  • og:type：常用 website / article。
  • 必要时补充 Twitter Card 以提升兼容性。

可使用各平台调试工具（如 Facebook Sharing Debugger、Twitter Card Validator）检查解析结果。
