# meta

**详解 meta 标签的使用**

----


## 概述
meta(Metadata) 用于控制 html 文档等一系列特性
更具浏览器厂商不同抱哈一系列特性。

## 核心属性
### charset
该属性决定 html 采用何种字符集进行解析.
对于解析完成的内容修改 `charset` 无效果

### name,content
用于描述文档,该描述可用于 SEO
