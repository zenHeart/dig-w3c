## DOM 是什么?
DOM 又叫文档对象模型，是 W3C 规定的一套用来操作 HTML 文档的 DEVICE_API 接口。
主要作用是允许利用 DEVICE_API 来修改 HTML 文档的结构和属性。具体包括如下部分。

* **DOM 对象**   
DOM 对象反应了 HTML 文档的结构，整个 HTML 文档被 DOM 对象以节点的方式展现出来。
